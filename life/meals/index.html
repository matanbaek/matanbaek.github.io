Section 3 — AI 식단 파이프라인

GPT-4o + 한국 식품 DB 결합 RAG

단순 "식단 짜줘"가 아니라 국내 식약처 영양소 DB·훈련 데이터를 조합한 구조화된 식단 설계입니다.

파이프라인

1. 입력훈련 부하·체성분·재고
2. 정규화JSON 스키마 변환
3. 영양 DB식약처 DB + 성분표 RAG
4. 식단 초안GPT-4o 7일 식단 생성
5. 검증PFC·미량 영양소 자동 체크
6. 장보기쿠팡 장바구니 스크립트

모델은 단순 조리법 나열을 피하기 위해 "하드데이 탄수 +30%, 회복일 단백 유지·탄수 -20%" 같은 정량 제약을 시스템 프롬프트에 박아두었습니다. 결과는 JSON으로 받아 Python 스크립트가 PFC 합계를 자동 검증합니다.

제약 1 · 알레르기갑각류 제외, 유제품 주 3회 이내
제약 2 · 예산주 10만원 · 외식 2회 허용
제약 3 · 조리 시간평일 조리 30분 이내
제약 4 · 체성분 목표체지방률 12% · 근육량 유지

회식은 대략 1,400~1,600kcal 예상(프라이드 300g + 맥주 500ml×3). 현명한 분배:
